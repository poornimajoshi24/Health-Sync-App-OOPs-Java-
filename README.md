# Health Sync App

Health Sync App is a **Java-based desktop application** developed using
**Object-Oriented Programming (OOPs)** concepts.  
The application helps users **track daily health activities** such as
exercise, medicine intake, water consumption, and reminders, and finally
displays **health reports** using a GUI built with **AWT and Swing**.

---

##  Feature

-  Daily exercise tracking  
-  Medicine intake management  
-   Water intake logging  
-  Health reminders  
-  Daily health report generation  
-  Graphical User Interface using AWT & Swing  

---

##  Technologies Used

- **Java**
- **Object-Oriented Programming (OOPs)**
- **AWT (Abstract Window Toolkit)**
- **Swing**
- **IntelliJ IDEA**

---

##  OOPs Concepts Implemented

- Classes and Objects  
- Encapsulation  
- Inheritance  
- Polymorphism  
- Abstraction
